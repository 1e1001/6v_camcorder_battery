# 6v_camcorder_battery
Replacement for batteries in a common camcorder form factor

I bought a $2 camcorder and wanted to make a battery that would be cheaper than that. It seems that many other camcorders use this form facter.

The contacts are any ~7mm wide strips of conductive metal - I used the shell of a dead AA battery to make mine. They might be too far from the contacts on the camera, I don't remember if I fixed that yet.

The battery-box tray is sized to the specific boxes I had available, and is designed to fit 5 AAs with an extra "null" battery, it will likely need modification for one's specific needs. The excess width somewhat gets in the way of the use of the camera, e.g. ejecting the tape.

If I have a want to improve the design in the future, I might try to make it using li-* batteries instead, with some built-in BMS instead of relying on the camera's.
